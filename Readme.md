## Front-End MVC Demo
### MVC Demo.
> Descripción: Solución VS con un proyecto que utiliza el patrón de diseño MVC: Modelo-Vista-Controlador.
+ Vista Home: Indice, Contacto y Acerca de.
+ Vista Languages: De acuerdo al parámetro (español o english) mostrará un título en inglés o español.
                   Si el parámetro no coindice entonces mostrará una página por defecto.
+ Vista Forms: Adicionar contactos, probar distintos eventos con javascript, cargar datos desde página asp y probar AjaX y AjaJ.
+ Vista People: CRUD que permite crear, listar items y ver detalles de personas. Se apoya de clase modelo Person.cs.
+ Vista SendEmail: Formulario con sus correspondientes tipos de validaciones y enviar correo electrónico mediante controladora.
                   Se apoya de clase modelo Email.cs.
+ EvalServiceSite: sitio web que expone el servicio WCF para ser consumido.
+ EvalWindowsService: servicio de Windows que expone el servicio.
